# Vision

## Problema

Muchas personas registran sus gastos de manera desordenada o simplemente no llevan control financiero. Las aplicaciones existentes suelen ser demasiado complejas, saturadas de funciones o poco intuitivas.

## Objetivo

Construir una aplicación web de finanzas personales simple, rápida y enfocada en el control diario de ingresos y gastos.

La aplicación permitirá:

- Registrar transacciones
- Clasificar gastos por categorías
- Visualizar balances
- Entender hábitos financieros básicos

## Usuario objetivo

Personas que desean comenzar a organizar sus finanzas personales de manera sencilla, sin conocimientos técnicos ni contables.

Principalmente:

- estudiantes
- trabajadores independientes
- empleados
- usuarios que nunca han usado apps financieras

## Filosofía del proyecto

El proyecto prioriza:

- simplicidad
- velocidad
- diseño limpio
- arquitectura escalable
- código mantenible

Se busca construir primero un MVP sólido antes de añadir funcionalidades avanzadas.
